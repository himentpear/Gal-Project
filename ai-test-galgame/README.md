# Gal-Project Web Version

This project includes a web-based visual novel engine inspired by Danganronpa's debate mechanics.

## How to Run

1.  Make sure you have Python installed.
2.  Run the following command from the root of the repository:

    ```bash
    python3 ai-test-galgame/src/main.py
    ```

3.  This will start a local web server and attempt to open your default web browser to the game page.
    If it doesn't open automatically, navigate to `http://localhost:8000` in your browser.

## Controls

*   **Dialogue**: Click anywhere to advance the text.
*   **Debate**: Click to advance. When you see yellow text (Weak Point), click specifically on the yellow text to break the argument and proceed to the resolution.

## Content

The current scenario "The Library Incident" features a debate between Yang Yichen and Xiao Jingcha.
